# CloudService (SaaS Платформа)

## Описание проекта

Прототип веб-приложения для облачной SaaS-платформы. Проект создан в рамках лабораторной работы по настройке инструментов веб-разработки.

## Используемые технологии

- HTML5, CSS3, JavaScript
- Node.js, npm
- Git, GitHub
- Figma
- VS Code с расширениями (Live Server, Prettier, GitLens)
